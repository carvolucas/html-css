Principais formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag <b> (não semântica)

mas nessa frese temos um termo em destaque usando a tag <STRONG> (semântica)

Itálico / Ênfase

Nessa frase temos um termo em itálico usando a tag <i> (não semântica)

mas nessa frase temos um termo em ênfase usando a tag <em> (semântica)

Marcação / marca-texto

Nessa frase temos um termo destacado com marcação usando a tag <mark>

Outro parágrafo temos um outro texto marcado usando a mesma tag <mark>

Texto grande e pequeno

Podemos deixar um texto um pouco maior ou também um pouco menor nesse parágrafo

Texto deletado

Podemos marcar um texto como excluído, para indicar que ele pode ser lido mas não considerado

Texto inserido

podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois

Texto sobrescrito

Para inserir coisas do tipo x20+3

Texto subscrito

Para inserir coisas do tipo H2O

Citações

podemos colocar uma citação dentro de um parágrafo escrito como uma frase de citação usando a tag <q>

Podemos criar citações mais longas, com um parágrafo só pra si, usando a tag <blockquote>

Abreviações

podemos criar abreviações de palavras usando a tag <abbr>
